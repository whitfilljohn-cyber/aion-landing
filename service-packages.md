- Are willing to pay for developer tools
- Hate subscriptions
- Value privacy

After they buy the $15 product, upsell to:
- $150 Quick Start ("Don't waste a weekend configuring it")
- $350 Codebase Intelligence ("Let me do the hard work")

- 100 people visit landing page → 3 buy $15 product (3%)

- 5 buy $150 service → 1 buys $350 package (20%)
- 1 buys $350 package → 1 signs up for $500/month (ongoing)

**Per 100 visitors: $15×3 + $150×5 + $350×1 + $500 = $1,395**

## Delivery Process

### For Tier 1 ($150):
1. Client pays via Stripe/PayPal
2. Send Calendly link for screenshare
3. 30-min Zoom call:
   - Share screen, install Python if needed
   - Install Aion
   - Configure API keys
   - Watch their project folder
   - Test with one command
4. Send follow-up email with:
   - Recording of session
   - Quick reference card
   - Support email

### For Tier 2 ($350):
1. Client fills intake form (repo URL, tech stack, pain points)
2. Clone their repo locally (or they screenshare)
3. Run Aion deep scan overnight
4. Generate documentation package
5. 60-min delivery call:
   - Walk through generated docs
   - Train on memory system
   - Show advanced features
   - Q&A
6. Deliver:
   - All generated documentation (Markdown files)
   - Aion config backup
   - 14-day support access

### For Tier 3 ($500/month):
1. Monthly retainer via Stripe subscription
2. Onboarding (same as Tier 2)
3. Weekly:
   - Run automated analysis
   - Send report via email
   - Flag 3 high-priority suggestions
4. Monthly:
   - 1-hour strategy call
   - Update model routing if needed
   - Add custom tool if requested
5. Ongoing Slack/Discord access
